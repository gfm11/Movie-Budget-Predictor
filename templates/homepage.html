Movie Statistics Predictor

Search for, update and analyze film records.

How to Use

To use the basic functions, click on the update records tab in the navigation bar, then find the form with the action you would like to perform. If you would like to insert a new movie, fill out the insert movie form. If you would like to update a movie, fill out the update movie form. If you would like to remove a movie, fill out the remove movie form. You must fill in the required fields in order for the movie to be inserted, updated, or removed.
